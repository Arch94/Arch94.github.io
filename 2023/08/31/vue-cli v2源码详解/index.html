vue-cli v2源码详解

总结

vue init <template-name> <project-name> 执行过程分析

  1. 获取参数
  2. 获取模板路径
  3. run 函数：区分本地和离线，下载模板

generate方法模板渲染的流程：

  1. 获取模板配置，
  2. 初始化 Metalsmith，添加变量至 Metalsmith
  3. handlebars 模板注册 helper
  4. 执行 before 函数(如果有)
  5. 询问问题，过滤文件，渲染模板文件
  6. 执行 after 函数(如果有)
  7. 构建项目
  8. 构建完成后，有 complete 函数则执行，没有则打印配置对象中的 completeMessage 信息，有错误就执行回调函数 done(err)
